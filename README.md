# AI Document Workflow Agent for Small Businesses

A production-style portfolio project for helping small businesses work with documents.

## Goal

Users will be able to upload business documents, ask questions with source citations, extract structured fields, generate summaries, create action items, and export useful results.

## Current Status

Phase 1 is complete: the Next.js app shell, TypeScript configuration, Tailwind setup, dashboard placeholder, document workspace placeholder, API health route, environment template, and project folders are in place.

## Recommended Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- OpenAI API
- Supabase Postgres, pgvector, Auth, and Storage
- Vercel deployment

## Local Development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
